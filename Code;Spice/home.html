Why join the Code;Spice program?

The Code;Spice program teaches people about the basics of Python and programming itself, which is useful to learn in this age. It is also very fun!

What is the cost to attending?

Code;Spice is completely free! Also, there isn't a huge time commitment necessary to this program. Our schedules are built to accomodate yours!

What can people expect from the program?

People can expect fun, interactive classes that are also instructive. They will be able to confidently make basic Python applications by the end of the course.
